# Sprint PRD: Web Sprint 04 - Hardening and Local Serve

> PRD tổng: [web-static-local-prd.md](../web-static-local-prd.md)

## 1. Mục tiêu sprint
- Ổn định web demo cho môi trường local, thêm hướng dẫn serve và checklist regression.

## 2. Phụ thuộc
- Hoàn tất [web-sprint-03-clone-export-states.md](./web-sprint-03-clone-export-states.md)

## 3. Phạm vi
- Ghi rõ cách serve local.
- Kiểm tra behavior khi CORS hoặc API fail.
- Rà lại copy, trạng thái, error banner, result panel.
- Hoàn thiện smoke checklist cho demo.

## 4. Deliverables
- Local serve notes rõ ràng.
- Demo flow ổn định cho synthesize và clone.
- Regression checklist cho QA/dev nội bộ.

## 5. Acceptance criteria
- Web chạy được qua local static server đơn giản.
- Không cần framework hoặc build tool.
- Error path không để UI rơi vào state kẹt.

## 6. Điều kiện hoàn thành
- Có tài liệu đủ để người khác mở web demo và test ngay với desktop API.
